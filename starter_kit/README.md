# Starter Kit

This folder contains runnable examples and lightweight helpers so students can
start the Reconnaissance Mission assignment without access to the full project.

## Contents
- `modules/`: minimal helper modules (Drone, Tower, World, Messaging)
- `examples/`: buildable/runnable example programs and launch notes
- `assets/`: sample arena and color data

## How to Use
1. Start with `examples/mpi_round_robin.cpp` to verify MPI setup.
2. Use `examples/mpi_struct_message.cpp` to see struct messaging with MPI.
3. Use `examples/gui_moving_dot.cpp` to verify GUI basics.
4. Use `examples/go_server/main.go` + `examples/cpp_to_go_client.cpp` for a simple telemetry pipeline.
5. Expand the modules as you implement the full assignment.
