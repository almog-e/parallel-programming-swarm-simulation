# Starter Kit Examples

## MPI Round Robin
- `mpi_round_robin.cpp`: passes a token through ranks in a ring.

## MPI Struct Message
- `mpi_struct_message.cpp`: sends a struct with position data from rank 0 to rank 1.

## GUI Moving Dot
- `gui_moving_dot.cpp`: minimal GLUT example that moves a dot on screen.

## Go Server + C++ Client
- `go_server/main.go`: TCP server that logs received lines.
- `cpp_to_go_client.cpp`: sends the numbers 1..5 to the Go server.
